# 🚀 Hướng Dẫn Cài Đặt và Chạy Dự Án

Để chạy dự án này, trước tiên bạn cần tạo môi trường ảo Python, kích hoạt nó và cài đặt các thư viện cần thiết.

Chạy lần lượt các lệnh sau trong Terminal hoặc CMD:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
